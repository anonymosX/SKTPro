changedomainhere works hard to make sure our customers are completely satisfied. We are a small family run company so we want to make sure you are happy. If an order is not completely as expected, we will issue a full refund.

How to Start a Free Return

1) To start a return please send us a message via the Email Us form or email us at changemailhere. Please be sure to include your order number, if possible. Please be sure that your return meets the Return Policy Requirements listed below.

2) Upon return authorization, you will receive a prepaid USPS label via email. Print label.

3) Package your item in its’ original box/bag and affix USPS label. Be sure to remove or cover up any old labels. Drop in mailbox and you’re done!

4) Please allow 5-7 days for return to be processed and payment refunded.

Damaged Package

If your package arrived damaged, we will send you a replacement right away at no additional cost to you. If you need to report a damaged package, please send us a message via the Email Us form or email us at changemailhere . Please be sure to include your order number, if possible.

Return Policy Requirements

changedomainhere offers a generous 30-day return policy. Refunds will be issued back to the original credit card used to make the purchase. Products must be returned in same condition in which they were received.

*Open products, probiotics and other refrigerated items cannot be returned.

* Shipping costs are non-refundable for undeliverable packages due to address error, refused delivery or unclaimed delivery.
